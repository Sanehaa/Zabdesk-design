Course Description

In this course basic concepts of calculus and three dimensional geometry will be covered.

Course objective

This course extends development of topics which include function and graph, limit and continuity, differentiation, integration and area between two curves.

Learning Outcomes

The student would model the problem related to computer science by using the principle of calculus. The student would be able to apply the calculus for solving the problems.

Teaching and Learning Methodology

Lectures, learning through problem solving, written assignments

Materials and Supplies

Marker, duster, computer, internet and projector

Maturity and punctuality
